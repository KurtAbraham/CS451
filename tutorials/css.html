Introduction

CSS stands for Cascading Style Sheets
CSS describes how HTML elements are to be displayed on screen, paper, or in other media
CSS saves a lot of work, controlling the layout of multiple web pages all at once
External stylesheets are stored in CSS files
Differing CSS files can make immense changes to the design, layout, and variations in display for same HTML file
Below is a sample CSS file that sets the style for the body, h1, and p tags:

body {
background-color: lightblue;
}

h1 {
color: white;
text-align: center;
}

p {
font-family: verdana;
font-size: 20px;
}

Syntax

A CSS rule consists of a selector and a declaration block, like so:

h1 {
color: blue;
font-size: 12px;
}

The selector (h1) points to the HTML element you want to style
The declaration blocks (color: blue; font-size: 12px;) contains declaration(s) ending with a semicolon
Each declaration includes a CSS property name and a value, separated by a colon
Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces
Below is a sample of a CSS file that will change the color of <p> text to red and align it to the center:

p {
color: red;
text-align: center;
}

Selectors

A CSS selector selects the HTML element(s) you want to style
CSS selectors can be divided into five categories:
1) Simple selectors (select elements based on name, id, class)
2) Combinator selectors (select elements based on a specific relationship between them)
3) Pseudo-class selectors (select elements based on a certain state)
4) Pseudo-elements selectors (select and style a part of an element)
5) Attribute selectors (select elements based on an attribute or attribute value)

The element selector selects HTML elements based on the element name, such as <p> in the following sample:

p {
text-align: center;
color: red;
}

The id selector uses the id attribute of an HTML element to select a specific element
The id of an element is unique within a page, so the id selector is used to select one unique element
To select an element based on its id, write a hash (#) character followed by the id, like so:

#para1 {
text-align: center;
color: red;
}

The class selector selects HTML elements within a specific class attribute
To select elements with a specific class, write a period (.) character followed by the class name, like so:

.center {
text-align: center;
color: red;
}

You can also specify that only specific HTML elements should be affected by a class, such as <p> elements with class center:

p.center {
text-align: center;
color: red;
}

The universal selector (*) selects all HTML elements on the page:

* {
text-align: center;
color: blue;
}

The grouping selector selects all HTML elements with the same style definitions
The sample code below sets the same style for the h1, h2, and p elements

h1, h2, p {
text-align: center;
color: red;
}

Comments

Comments in CSS files follow the same rules as in HTML files
Comments can be placed anywhere within the code, and will be ignored by web browsers
A CSS comment starts with /* and ends with */, like so:

p {
color: red; /* Set text color to red */
}

Colors

Colors can be specified using predefined color names, RGB, HEX, HSL, RGBA, or HSLA values
HTML/CSS supports 140 standard color names, such as Orange, MediumSeaGreen, and Violet
You can set the background color of HTML elements like so:

<h1 style="background-color:DodgerBlue;">Hello World</h1> <p style="background-color:Tomato;">Lorem ipsum...</p>

You can set the color of text like so:

<h1 style="color:Tomato;">Hello World</h1> <p style="color:DodgerBlue;">Lorem ipsum...</p> <p style="color:MediumSeaGreen;">Ut wisi enim...</p>

You can set the color of borders like so:

<h1 style="border:2px solid Tomato;">Hello World</h1> <h1 style="border:2px solid DodgerBlue;">Hello World</h1> <h1 style="border:2px solid Violet;">Hello World</h1>

Backgrounds

The CSS background properties are used to add background effects for elements
The background color of a page is set like so:

body {
background-color: lightblue;
}

The opacity property specifies the opacity/transparency of an element
It can take a value between 0.0 and 1.0
The lower the value, the more transparent:

div {
background-color: green;
opacity: 0.3;
}

You can also specify the opacity of the background using RGBA, like so:

div {
background: rgba(0, 128, 0, 0.3) /* Green background with 30% opacity */
}

Borders

The CSS border properties allow you to specify the style, width, and color of an element's border
The border-style property specifies what kind of border to display, such as:
1) dotted - Defines a dotted border
2) dashed - Defines a dashed border
3) solid - Defines a solid border
4) double - Defines a double border
5) hidden - Defines a hidden border
The border-style property can have one to four values, each specifying the top, right, bottom, and left borders like so:

p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}
p.mix {border-style: dotted dashed solid double;}

Margins

Margins are used to create space around elements, outside of any defined borders
CSS has properties that specify the margin for each side of an element:
1) margin-top
2) margin-right
3) margin-bottom
4) margin-left

All the margin properties can have the following values:
1) auto - the browser calculates the margin
2) length - specifies a margin in px, pt, cm, etc.
3) % - specifies a margin in % of the width of the containing element
4) inherit - specifies that the margin should be inherited from the parent element
Negative values are allowed!
Below is a sample of different margins for all four sides of a <p> element:

p {
margin-top: 100px;
margin-bottom: 100px;
margin-right: 150px;
margin-left: 80px;
}

If the margin property has four values, it will set the top, right, bottom, and left margins respectively:

p {
margin: 25px 50px 75px 100px;
}

If the margin property has three values, it will set the top, right AND left, and the bottom margins respectively

p {
margin: 25px 50px 75px;
}

If the margin property has two values, it will set the top AND bottom, and the right AND left margins respectively

p {
margin: 25px 50px;
}

If the margin property has one value, it will set all margins

You can set the margin property to auto to horizontally center the element within its container
The element will take up the specified width, then the remaining space will be split evenly between the left and right margins, like so:

div {
width: 300px;
margin: auto;
border: 1px solid red;
}

Padding

Padding is used to create space around an element's content, inside of any defined borders
CSS has properties for specifying the madding for each side of an element:
1) padding-top
2) padding-right:
3) padding-bottom:
4) padding-left:
Padding can have the following values:
1) length - specifies a padding in px, pt, cm, etc.
2) % - specifies a padding in % of the width of the containing element
3) inherit - specifies that the padding should be inherited from the parent element
Negative values are not allowed for padding

Below is a sample of padding:

div {
padding-top: 50px;
padding-right: 30px;
padding-bottom: 50px;
padding-left: 80px;
}

Padding can have up to four values for each side, just like margins:

div {
padding: 25px 50px 75px 100px;
}

Padding adds extra width to elements, so the following sample has a div element width of 350px
The width of the element itself is 300px, but the padding of 25px adds the extra 50px width:

div {
width: 300px;
padding: 25px;
}

Height/Width

The CSS height and width properties are used to set the height and width of an element
Height and width can have the following values:
1) auto - This is default. The browser calculates the height and width
2) length - Defines the height/width in px, cm, etc.
3) % - Defines the height/width in percent of the containing block
4) initial - Sets the height/width to its default value
5) inherit - The height/width will be inherited from its parent value

The following sample element has a set height of 200px and a width of 50% of the containing block:

div {
height: 200px;
width: 50%;
}

Height and width DO NOT include padding, borders, and margins!
Instead, height and width set the area inside padding, borders, and margins

The max-width property can set the maximum width of an element
The max width enables the element to change its width when the size of the browser window is changed, like so:

div {
max-width: 500px;
height: 100px;
background-color: powderblue;
}

Box Model

The term "box model" is used when talking about design and layout
The box model is a box that wraps around every HTML element:
1) Content - The content of the box, where text and images appear
2) Padding - Clears an area around the content. The padding is transparent
3) Border - A border that goes around the padding and content
4) Margin - Clears an area outside the border. The margin is transparent
Below is a sample of an element that contains content, padding, border, and margin:

div { width: 300px;
border: 15px solid green;
padding: 50px;
margin: 20px;
}

The total width and height of an element is affected by the box model
The sample above has a width of 470px total:
Width of 300px + border of 30px + padding of 100px + margin of 40px = 470px

Outline

An outline is a line drawn outside the element's border to make the element stand out
Outline has the following properties:
1) outline-style
2) outline-color
3) outline-width
4) outline-offset
5) Outline
Unlike borders, outlines may overlap with other elements

1) The outline-style property specifies the style of the outline, and can have one of the following properties:
2) dotted - Defines a dotted outline
3) dashed - Defines a dashed outline
4) solid - Defines a solid outline
5) double - Defines a double outline
6) groove - Defines a 3D grooved outline
7) ridge - Defines a 3D ridged outline
8) inset - Defines a 3D inset outline
9) outset - Defines a 3D outset outline
10) none - Defines no outline
11) hidden - Defines a hidden outline
The following sample shows different outline-style values:

p.dotted {outline-style: dotted;}
p.dashed {outline-style: dashed;}
p.solid {outline-style: solid;}
p.double {outline-style: double;}
p.groove {outline-style: groove;}
p.ridge {outline-style: ridge;}
p.inset {outline-style: inset;}
p.outset {outline-style: outset;}

Text

CSS can format text in numerous ways!

The color property is used to set the color of text

body {
color: blue;
}

h1 {
color: green;
}

The text-align property is used to set the horizontal alignment of text
Text can be left or right aligned, centered, or justified

The text-align-last property specifies how to align the last line of text, like so:

p.a {
text-align-last: right;
}

p.b {
text-align-last: center;
}

p.c {
text-align-last: justify;
}

The direction and unicode-bidi properties can be used to change the text direction of an element, like so:

p {
direction: rtl;
unicode-bidi: bidi-override;
}

The vertical-align property sets the vertical alignment of an element, like so:

img.a {
vertical-align: baseline;
}

img.b {
vertical-align: text-top;
}

img.c {
vertical-align: text-bottom;
}

img.d {
vertical-align: sub;
}

img.e {
vertical-align: super;
}

The text-decoration-line property is used to add a decoration line to text
Text decoration lines can have their color, style, and thickness specified with the following properties:
1) text-decoration-color
2) text-decoration-style
3) text-decoration-thickness
Below is a sample of text that has text decoration lines using all of the properties:

p {
text-decoration-line: underline;
text-decoration-color: red;
text-decoration-style: double;
text-decoration-thickness: 5px;
}

The text-transform property is used to specify uppercase and lowercase letters in a text
It can turn everything into UPPERCASE, lowercase, or capitalize the first letter of each word like so:

p.uppercase {
text-transform: uppercase;
}

p.lowercase {
text-transform: lowercase;
}

p.capitalize {
text-transform: capitalize;
}

The text-indent property specifies the indentation of the first line of text:

p {
text-indent: 50px;
}

The letter-spacing property is used to specify the space between the characters in a text, and can be negative:

h1 {
letter-spacing: 5px;
}

h2 {
letter-spacing: -2px;
}

The line-height property specifies the space between lines:

p.small {
line-height: 0.8;
}

p.big {
line-height: 1.8;
}

The word-spacing property specifies the space between the words in a text:

p.one {
word-spacing: 10px;
}

p.two {
word-spacing: -2px;
}

The white-space property specifies how white-space inside an element is handled:

p {
white-space: nowrap; }

The text-shadow property adds shadow to text
By default, text shadows are 2px by 2px
Text shadows can be blurred, colored, and overlapped with other shadows like so:

h1 {
color: white;
text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue;
}

Fonts

Using the right fonts for websites are very important
The font-family property specifies the font of a text
The font-family property can hold several font names as a backup in case other fonts are incompatible with some browsers/operating systems
The leftmost font takes priority over the fonts to the right
Below is a sample of different fonts that are being specified for three paragraphs:

.p1 {
font-family: "Times New Roman", Times, serif;
}

.p2 {
font-family: Arial, Helvetica, sans-serif;
}

.p3 {
font-family: "Lucida Console", "Courier New", monospace;
}

The font-size property sets the size of the text
The font size can be set to pixels
Font size can also be set to units of em, which allows users to resize the text in the browser menu
1em is equal to the current font size, and the default font size in browsers is 16px
A good rule of thumb is to always use em instead of pixels!
Below is a sample of fonts set to em followed by size calculations:

h1 {
font-size: 2.5em; /* 40px/16=2.5em */
}

h2 {
font-size: 1.875em; /* 30px/16=1.875em */
}

p {
font-size: 0.875em; /* 14px/16=0.875em */
}

Text size can also be set with vw units, which means "viewport width"
The size of text will follow the size of the browser window, like so:

<h1 style="font-size:10vw">Hello World</h1>

Links

Links can be styled with any CSS property just like regular text
In addition, links can change style depending on what state they are in:
1) a:link - a normal, unvisited link
2) a:visited - a link the user has visited
3) a:hover - a link when the user mouses over it
4) a:active - a link the moment it is clicked
Below is a sample of stylings for the four different states of links:

/* unvisited link */
a:link {
color: red;
}

/* visited link */
a:visited {
color: green;
}

/* mouse over link */
a:hover {
color: hotpink;
}

/* selected link */
a:active {
color: blue;
}

There are some rules when setting styles for several link states:
1) a:hover MUST come after a:link and a:visited
2) a:active MUST come after a:hover

The text-decoration property can be used to remove underlines from links like so:

a:link {
text-decoration: none;
}

a:visited {
text-decoration: none;
}

a:hover {
text-decoration: underline;
}

a:active {
text-decoration: underline;
}


Below is a sample that demonstrates an advanced method to display links as buttons or boxes:

a:link, a:visited {
background-color: #f44336;
color: white;
padding: 14px 25px;
text-align: center;
text-decoration: none;
display: inline-block;
}

a:hover, a:active {
background-color: red;
}

Position

The position property specifies the type of positioning method used for an element
The methods that can be used include static, relative, fixed, absolute, and sticky

An element with position: static; is not positioned in any special way
Instead, it is positioned according to the normal flow of the page

An element with position: relative; is positioned relative to its normal position
Other content will not be adjusted to fit into any gap left by the element

An element with position: fixed; is positioned relative to the viewport
It will not leave a gap where it would normally be located
It will stay in the same place even if the page is scrolled

An element with position: absolute; is positioned relative to the nearest positioned ancestor
If there are no positoned ancestors, it uses the document body and moves along with page scrolling
Absolute positioned elements can overlap elements and are removed from the normal flow of the webpage

An element with position: sticky; is positioned based on the user's scroll position
A sticky element toggles between relative and fixed depending on the scroll position
In other words, the element will scroll inside a container and "stick" to the edges of the container

Below is a compilation of the properties mentioned above:

div.static {
position: static;
border: 3px solid #73AD21;
}

div.relative {
position: relative;
left: 30px;
border: 3px solid #73AD21;
}

div.fixed {
position: fixed;
bottom: 0;
right: 0;
width: 300px;
border: 3px solid #73AD21;
}

div.absolute {
position: absolute;
top: 80px;
right: 0;
width: 200px;
height: 100px;
border: 3px solid #73AD21;
}

div.sticky {
position: -webkit-sticky; /* Safari */
position: sticky;
top: 0;
background-color: green;
border: 2px solid #4CAF50;
}

Float

The CSS float property is used to specify the positioning and formatting of content
For example, letting an image float to the left of text in a container
The float property can have the following values:
1) left - The element floats to the left of its container
2) right - The element floats to the right of its container
3) none - The element does not float (will be displayed just where it occurs in the text). This is default
4) inherit - The element inherits the float value of its parent

Normally, div elements will be displayed on top of each other
If we use float: left;, we can let elements float next to each other
Below is a sample of the float property:

img {
float: left;
}

div {
float: left;
padding: 15px;
}

.div1 {
background: red;
}

.div2 {
background: yellow;
}

.div3 {
background: green;
}

Align

There are many ways to align content on a webpage, and below are some examples of how to do so!

To align a block element to the center horizontally, use margin: auto;
This works because setting the width of the element will prevent it from stretching out to the edges of its container
The element will take up the specified width, then the remaining space will be distributed equally to the remaining margins, like so:

.center {
margin: auto;
width: 50%;
border: 3px solid green;
padding: 10px;
}

To center text inside an element, just use text-align: center;

To center an image, set the left and right margin to auto and make it into a block element, like so:

img {
display: block;
margin-left: auto;
margin-right: auto;
width: 40%;
}

For more examples, visit w3schools.com

Opacity

The opacity property specifies the opacity/transarency of an element
Opacity can take on a value from 0.0 - 1.0
The lower the value, the more transparent the element will be
The sample below shows an image that is 50% transparent

img {
opacity: 0.5;
}

The opacity property is often used with the :hover selector to change the opacity on mouse-over

If you use opacity to add transparency to the background of an element, all its child elements will inherit the same transparency
This can be fixed by using RGBA color values to set the background color, which will not affect its child elements:

div {
background: rgba(76, 175, 80, 0.3) /* Green background with 30% opacity */
}
