Introduction

HTML stands for "Hyper Text Markup Language"
HTML is the standard markup language for making websites
HTML tells web browsers how to display various content

A sample HTML document is shown below:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>Sample Heading</h1>
<p>Sample paragraph</p>

</body>
</html>

Here is an explanation of the sample HTML document:
The <!DOCTYPE html> declaration defines that this document is an HTML5 document
The <html> element is the root element of an HTML page
The <head> element contains meta information about the HTML page
The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
The <h1> element defines a large heading
The <p> element defines a paragraph

Basics

All HTML documents must start with a document type declaration: <!DOCTYPE html>
The HTML document itself begins with <html> and ends with </html>.
The visible part of the HTML document is between <body> and </body>

HTML headings are defined with the <h1> to <h6> tags
<h1> defines the most important heading. <h6> defines the least important heading
Some sample headers are listed below:

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>

HTML paragraphs are defined with the <p> tag

HTML links are defined with the <a> tag, like so:

<a href="https://www.w3schools.com">This is a link to W3Schools</a>

HTML images are defined with the <img> tag
The source file (src), alternative text (alt), width, and height are provided as attributes, like so:

<img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">

Elements

HTML elements are everything from the start tag to the end tag

<tagname>Content goes here</tagname>

HTML elements can be nested, or in other words, contained within other elements
The following sample HTML document utilizes elements nested within the body, which is nested within the html element:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>Sample Heading</h1>
<p>Sample paragraph</p>

</body>
</html>

A good rule of thumb is to NEVER forget the end tag!
Errors may occur without end tags

However, some elements are empty and do not require an end tag
An example is the <br> tag, which stands for a line break, like so:

<p>This is a <br> paragraph with a line break.</p>

Attributes

All HTML attributes can have attributes
Attributes provide additional information about elements, like arguments
Attributes are specified in the start tag, and typically come in name/value pairs

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to, like so:

<a href="https://www.w3schools.com">Visit W3Schools</a>

The <img> tag is used to embed an image in an HTML page. The src attribute specified the path to the image to be displayed, like so:

<img src="img_painting1.jpg">

The <img> attribute can also contain an alt attribute, which specifies alternate text in case the image cannot be displayed, like so:

<img src="img_painting1.jpg alt="Painting of the Mona Lisa">

The title attribute defines extra information about an element when it is hovered over with the mouse:

<p title="It's just sample text">This is some sample text</p>

A good rule of thumb is to ALWAYS put quotation marks around attribute values!
Errors may occur without quotation marks
You may use 'single quotes' or "double quotes," and in some cases, both will be required

Comments

Comments are not displayed in the web browser, but can help document HTML source code
HTML comments are added using the following syntax:

<!-- Comments go here -->

An exclamation point is needed in the start tag, but not the end tag

Comments can be used to provide reminders for future editors of the program
Comments can also be used to hide content temporarily from the web browser, like so:

<!--
<p>This paragraph is currently hidden</p>
-->

Comments can hide parts in the middle of HTML code on one line, like so:

<p>I <!-- truly do --> hope you understand!<p>

Links

HTML links are hyperlinks
You can click on a link to jump to another document
When you hover over a link, the mouse arrow will turn into a pointing hand

The HTML <a> tag defines a hyperlink and has the following syntax:

<a href="url">link text</a>

The href attribute indicates the link's destination
The link text is the part that will be visible to the end users
clicking on the link text will send the user to the specified URL address

An absolute URL addresses a full web address (using the "https://www" part)
A relative URL addresses a local link (without using the "https://www" part)
The sample links below show the difference between absolute and relative URLs respectively

<h2>Absolute URLs</h2>
<p><a href="https://www.w3.org/">W3C</a></p>
<p><a href="https://www.google.com/">Google</a></p>

<h2>Relative URLs</h2>
<p><a href="html_images.asp">HTML Images</a></p>
<p><a href="/css/default.asp">CSS Tutorial</a></p>

You can use links to open the user's email program to send an email to a specified address:

<a href="mailto:someone@example.com">Send email</a>

Images

Images can greatly improve the design and appearance of the webpage
The <img> tag is empty and contains attributes only, and thus does not need an end tag
It does, however, have two required attributes:
1) src - Specifies the path to the image
2) alt - Specifies an alternate text for the image

The syntax for images are as follows:

<img src="url" alt="alternatetext">

If a website's images are stored in another subfolder, you must include the folder name in the src attribute:

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

If a website uses an image from another server, the absolute URL must be specified in the src attribute:

<img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3Schools.com">

Images can be used as a link by putting the <img> tag inside the <a> tag:

<a href="default.asp">
<img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">
</a>

File Paths

A file path describes the location of a file in a website's folder structure
File paths are used when linking to external files, such as:
1) Web pages
2) Images
3) Style sheets
4) JavaScripts
An absolute file path is the full URL to a file:

<img src="https://www.w3schools.com/images/picture.jpg" alt="Mountain">

A relative file path points to a file relative to the current page
The following sample points to a file in the images folder located at the root of the current webpage:

<img src="/images/picture.jpg" alt="Mountain">

A good rule of thumb is to use relative file paths as much as possible!
While using relative file paths, web pages will not be bound to the current base URL
All links will work locally
