Python Tutorial

Introduction

Python is a powerful and versatile programming language known for its simplicity and readability. This tutorial will introduce you to the fundamental concepts of Python, allowing you to begin writing basic programs.

Setting Up

To start coding in Python, you'll need to install the Python interpreter. Download and install it from the official website: https://www.python.org/downloads/.

Hello, World!

Let's begin with the classic "Hello, World!" program. This simple program prints the message "Hello, World!" to the console.

    python
        print("Hello, World!")

Save this code as a file with a `.py` extension (e.g., hello.py) and run it using the Python interpreter. You should see the message displayed on the console.

Variables

Variables are used to store data in your programs. You can assign values to variables using the assignment operator (`=`).

    python
        name = "Alice"
        age = 30

        print("My name is", name, "and I am", age, "years old.")

In this example, we define two variables: `name` and `age`. We then use them in the `print` statement to display their values.

Data Types

Python supports various data types, such as:

  • **Strings:** Text data enclosed in quotes (single or double)
  • **Integers:** Whole numbers (e.g., 10, -5)
  • **Floats:** Numbers with decimal points (e.g., 3.14, -2.5)
  • **Booleans:** True or False values

Comments

Comments are lines of code ignored by the interpreter but serve as explanations for human readers. Use the `#` symbol to add comments.

    python
        # This is a comment
        print("This line will be executed")

Operators

Python provides various operators for performing calculations and comparisons:

  • **Arithmetic operators:** `+`, `-`, `*`, `/`, `%` (modulo)
  • **Comparison operators:** `==`, `!=`, `>`, `<`, `>=`, `<=`
  • **Logical operators:** `and`, `or`, `not`

Control Flow

Control flow statements allow you to control the execution order of your program.

  • **`if` statements:** Execute code conditionally based on a boolean expression.
  • **`for` loops:** Repeatedly execute a block of code for a certain number of times.
  • **`while` loops:** Repeatedly execute a block of code as long as a condition is true.

Functions

Functions are reusable blocks of code that perform specific tasks. You can define and call functions to organize your code and improve readability.

Further Learning

This is just a brief introduction to Python. As you progress, you can explore more complex concepts and utilize various libraries and frameworks to build powerful applications. Visit the official Python documentation for in-depth information: https://docs.python.org/.
