PHP Basics in One Page

What is PHP?

PHP (Hypertext Preprocessor) is a server-side scripting language for creating dynamic and interactive web pages. It allows developers to process information on the server before sending the resulting HTML content to the user's browser.

Basic PHP Syntax

PHP code is embedded within HTML using special tags: `<?php ... ?>`. Here are some key elements:

  • **Comments:** Explain code and are ignored by the server (e.g. `// This is a comment` or `/* Multi-line comment */`).
  • **Variables:** Store information (e.g. `$message = "Hello World!";` - variable name starts with `$`).
  • **Echo Statements:** Output content (e.g. `echo $message;` - displays the value stored in the variable).

**Example:** Displaying the current date and time:

<?php
    echo date("Y/m/d H:i:s");
    ?>

Variables and Data Types

Variables store information used throughout your code. They have a name and a data type (e.g., string, integer).

  • **Declaring Variables:** Use `$` followed by a descriptive name (e.g., `$firstName = "John";`).
  • **Data Types:** Common types include string (text), integer (whole numbers), float (decimal numbers), and boolean (true/false).

**Example:** Assigning values to different data types:

<?php
    $name = "Alice";
    $age = 30;
    $isMember = true;
    ?>

Operators in PHP

Operators perform calculations or comparisons on data. Common types include:

  • **Arithmetic:** Addition (+), subtraction (-), multiplication (*), division (/), etc.
  • **Comparison:** Equal to (==), not equal to (!=), greater than (>), etc.
  • **Logical:** AND (&&), OR (||), NOT (!).

**Example:** Checking if a user is eligible to vote (age >= 18):

<?php
    $age = 25;
    if ($age >= 18) {
      echo "You are eligible to vote.";
    } else {
      echo "You are not eligible to vote.";
    }
    ?>
